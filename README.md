# 💰 Finance Dashboard

## 📌 Overview
This is a simple and interactive Finance Dashboard built using HTML, CSS, and JavaScript. The application helps users track their income and expenses, view financial summaries, and understand spending patterns through visualizations.

---

## 🎯 Objective
The goal of this project is to design a clean and user-friendly interface that allows users to:
- Monitor financial activity
- Explore transactions
- Gain insights from data

---

## 🚀 Features

### 📊 Dashboard Overview
- Displays summary cards:
  - Total Income
  - Total Expense
  - Balance
- Provides a quick snapshot of financial status

---

### 📈 Data Visualization
- Time-based chart to represent financial trends
- Category-based chart for expense breakdown

---

### 📋 Transactions Management
- View all transactions with:
  - Date
  - Amount
  - Category
  - Type (Income/Expense)
- Add new transactions (Admin only)
- Edit existing transactions
- Delete transactions

---

### 🔍 Filtering & Search
- Filter transactions by:
  - All
  - Income
  - Expense
- Search transactions by category

---

### 👤 Role-Based UI
- **Viewer**
  - Can only view data
- **Admin**
  - Can add, edit, and delete transactions

---

### 📊 Insights
- Displays highest spending category
- Helps users understand spending behavior


## 🛠️ Technologies Used

- HTML (Structure)
- CSS (Styling & Responsive Design)
- JavaScript (Logic & State Handling)
- Chart.js (Data Visualization)
